WPML integration to show recents posts per language.

== Description ==

This plugin shows your recent posts per language via a widget with integration with WPML. If you enable this plugin you will see your posts with a thumbnail and a title in the language you are visiting the site.

== Installation ==

1. Upload `wpml-recents-posts-per-language.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
